Overview

Purpose

The purpose of this site is to market and sell digital sourdough recipe books for each season of the year. It will also be to provide an introduction to sourdough baking.

Audience

Home bakers and cooking enthusiasts looking for seasonal recipes. Primarily those who enjoy baking with sourdough or those who would like to learn how to bake with sourdough.

Dynamic elements

I plan on using a search and filter function on the home page for the recipe books that will be held in a js array. I will use a form for users to input data and submit it with conditional logic so it is filled out properly. I will also use flex box layout and media queries for a responsive design.
